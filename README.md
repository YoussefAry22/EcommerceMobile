### Frontend (Ionic-React) Setup
Navigate to the frontend directory:

   >cd ../frontend

¤ Install dependencies:

  >npm install

¤ Start the Ionic server:
  
  >ionic serve

### Backend (Springboot) Setup
Run a sql server ( like: wamp)
Run the project
